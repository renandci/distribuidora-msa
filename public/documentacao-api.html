Alterando estoques

Para alterar o produto com variação, envie um PUT /v1/produtos, seguindo a lógica abaixo.

O exemplo abaixo também pode ser usado para alterar preços, código de produtos, código de referência e postagem.

				"variations": [
					{

						"estoque": 5

						"id": 2628,
						"estoque": 1

					{

						"codigo_produto": "AA"
					}
				]
			}

Excluindo um produto com variação

Para excluir um produto com sua variação, ou o grupo envie um DELETE /v1/produtos, { "id": 5567 } específico ou {"codigo_id": 1234567890} para o grupo seguindo a lógica abaixo.
NOTA: Para aterar os dados, os id. id_* devem estar preenchidos. 

{"codigo_id": 1234567890}
{"id": 5567}

return 

{ "msg": "success" }
